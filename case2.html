About

Flowpay is a fintech company that provides online financing for enterprises. I was responsible for redesigning their product’s user interface and developing a custom design system tailored to their specific needs, building on top of an existing foundation. One of the key challenges was working with a complex, data‑intensive user interface that required thoughtful structuring and prioritization to maintain clarity. Additionally, the visual design was outdated, which is an important issue for a fintech platform where trust and credibility are essential. The result was a modern, scalable, and trustworthy interface that supports both business goals and user needs.

Challenges

  • Building a design system upon an existing system
  • Redesigning a complex, data‑intensive interface with a modern visual style to build user trust
  • Improving user experience across the product
  • Integrating accessibility principles into the design system
  • Addressing edge cases
  • Ensuring smooth collaboration and handoff with development teams

Results

  • Created a custom design system tailored to Flowpay's needs
  • Introduced a modern, professional visual style to enhance trust and brand perception
  • Enhanced consistency and accessibility by implementing best UX/UI practices
  • Improved usability across the platform, especially in high-density screens
  • Facilitated efficient design‑development collaboration with well‑structured design assets
