About

Flowpay is a fintech company that provides online financing for enterprises. I developed a custom design system tailored to their needs, building upon an existing system, and redesigned their product's UI. My work focused on improving the user experience by applying user‑centered design principles and enhancing usability.

My Role

UI/UX Designer
Design Systems Designer

Duration

Challenges

  • Building a design system upon an existing system
  • Developing a new user interface and visual style
  • Improving user experience across the product
  • Integrating accessibility principles into the design system
  • Addressing edge cases
  • Handoff of design assets for Development

Results

  • Created a custom design system tailored to Flowpay's needs
  • Created a cohesive visual style and a consistent user interface
  • Improved the color palette for better consistency and accessibility
  • Enhanced user experience by improving usability, navigation, and overall interface clarity
